# SYNTHEX SX-16 — Web Audio Synthesizer & Step Sequencer

A browser-based polyphonic step sequencer built with vanilla JavaScript and the Web Audio API.

## 🎵 Live Demo
[Launch Sequencer →](https://sejal-sisodia08.github.io/synthex-sequencer)

## ✨ Features
- 4x16 step sequencer grid with 4 instrument voices
- Real-time oscilloscope visualizer
- Waveform selector — Sine, Square, Sawtooth
- Volume and BPM controls
- Percussive sound engine with memory-safe audio cleanup

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript
- Web Audio API

## ⚙️ How to Run Locally
1. Clone this repository
2. Open `index.html` in Chrome or Firefox
3. No installations needed
